# handy-go

Atari Lynx (Handy) emulator core, packaged as a standalone submodule for
[game-and-watch-retro-go-sd](https://github.com/jshsakura/game-and-watch-retro-go-sd).

Source: extracted from [ducalex/retro-go](https://github.com/ducalex/retro-go)
`retro-core/components/handy` (K. Wilkins' Handy, Zlib license — see `license.txt`).

The G&W (STM32H7) porting/glue layer lives in the parent project under
`Core/Src/porting/lynx/`. This repo contains only the platform-agnostic emulator core.
